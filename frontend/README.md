# Frontend Documentation for the English Learning System

## Overview
This project is a web application designed to facilitate English language learning. It consists of a backend that handles data processing and a frontend that provides a user-friendly interface for learners.

## Project Structure
The frontend is organized into several key directories:

- **src/**: Contains the source code for the frontend application.
  - **components/**: Reusable UI components that can be used across different pages.
  - **pages/**: Contains the main pages of the application, each representing a different view.
  - **services/**: Contains services for making API calls and handling business logic.
  - **App.tsx**: The main component that sets up the application structure.
  - **index.tsx**: The entry point for the frontend application, responsible for rendering the App component.

## Getting Started
To get started with the frontend application, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd aprendizaje_ingles_web/frontend
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the application**:
   ```
   npm start
   ```

## Features
- Interactive UI for engaging with English learning materials.
- API integration with the backend for dynamic content.
- Responsive design for accessibility on various devices.
